Варіант 2. Бугай Олександр Юрійович
Каталог студентських робіт

Постановка задачі.
В університеті студенти виконують 2 види робіт: курсові роботи і
дипломні роботи (бакалавра і магістра).
Реалізувати 2 версії програми для роботи з каталогом студентських робіт
(зберігання відомостей про дипломні і курсові роботи).
Вимоги до програми.
1. Розробити структуру вхідного файлу або бази даних для зберігання
даних про роботи.
Для курсової роботи у файлі потрібно зберігати таку інформацію:
- назва дисципліни;
- тема курсової роботи;
- ПІБ студента;
- ПІБ керівника (викладача);
- група;
- рік захисту;
- оцінка.
Для дипломної роботи:
- кваліфікаційний рівень роботи ={бакалавр, магістр};
- тема дипломної роботи
- ПІБ студента;
- ПІБ керівника (викладача);
- група;
- рік захисту;
- оцінка.
Примітка.
Для того, щоб при читанні розрізняти курсові і дипломні роботи у файл
або табліцю БД внести ознаку роботи.
2. Розробити ієрархію класів (1 базовий, 2 похідних). У базовому класі
зберігати спільну інформацію про роботу, у похідних – специфічну (один
похідний клас для дипломних робіт, інший для курсових). Оголосити в
базовому класі віртуальний метод, який реалізувати у похідних класах.
3. У консольній версії створити одну або дві колекції List<> для
зберігання даних про роботи в пам’яті і заповнити їх з файлу або з бази даних.
4. Реалізувати функції програми:
- Реєстрація нових робіт (додавання у колекцію).
- Редагування інформації про курсові і дипломні роботи.
- Перегляд усіх назв тем курсових робіт за абеткою.
- Відбір і перегляд тільки курсових робіт.
- Перегляд назв тем магістерських робіт, які захищені у вказаному діапазоні років (або вказаному році).
- Пошук роботи за прізвищем студента.
- Перегляд робіт вказаного керівника.
- Видалення робіт з колекції.
5. У проекті з інтерфейсом Windows Forms використати колекцію
LinkedList<T> або відповідні класи EF Core. Реалізувати на формі перегляд
елементів колекції у прямому і зворотному напрямах. Реалізувати додавання,
видалення, редагування даних про роботи. Обчислити кількість захищених
дипломів у вказаному році.
